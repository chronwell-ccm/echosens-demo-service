# echosens x chronwell test API

Test/mock implementation of the echosens API for chronwell
This is using [pnpm](https://pnpm.io) as a package manager.

### Install

```
pnpm i  
```

### Develop

```
pnpm run dev
```

## build

```
pnpm run build 
docker build -t echosens-demo-service --progress=plain .    
```
